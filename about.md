# About This Site

This site collects tutorials, walkthroughs, and videos to help new members of the lab get oriented quickly — starting with humanoid robot motion control (Unitree G1) and expanding as the lab grows.

## Where to Start

If you're brand new, start with the "Getting Started" tagged posts, then move on to specific hardware and software guides.
